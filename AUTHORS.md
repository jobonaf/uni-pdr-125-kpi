## Come contribuire

Il software è rilasciato sotto licenza **EUPL v1.2** in conformità alle
Linee Guida AGID sull'acquisizione e riuso di software per le PA
(art. 69 D.Lgs. 82/2005).

Contribuzioni (bugfix, miglioramenti, traduzioni) sono benvenute tramite
pull request sul repository ufficiale. Si prega di rispettare le linee
guida di contribuzione nel file `CONTRIBUTING.md`.
